## **Using 📦**

1. Clone Template

```
git clone https://github.com/igdev116/vite-react-ts-eslint-prettier.git
```

2. Install Packages

```
yarn install
```

3. Start Project

```
yarn dev
```

4. If you using git, delete the existing folder .git after cloning (open `git bash` or other terminal)

```
rm -rf .git
```

## **Options ✍️**

1. Check lint

```
yarn lint
```

2. Fix lint

```
yarn lint:fix
```

3. Check prettier

```
yarn prettier
```

4. Fix prettier

```
yarn prettier:fix
```

5. Fix lint and prettier

```
yarn format
```
